# Clipboard Sync Magisk 模块 v1.3.0

无需 APK、Python、curl 或 wget。模块使用 Android 自带运行环境监听剪贴板变化，本机剪贴板变化时才上传；屏幕关闭时停止接收检查。

## 推荐安装

1. 登录 Clipboard Sync 网页并打开“设备与令牌”。
2. 填写手机名称，下载网页生成的版本化 Magisk ZIP。
3. 不要解压，在 Magisk 的“从本地安装”中选择 ZIP。
4. 重启手机。服务器地址和专属令牌已经写入包内，无需手工填写。

上传成功时会显示“已上传：内容预览”，收到其他设备内容时会显示“已接收（设备名）：内容预览”。预览会合并换行并截取前 40 个字符；如需保护屏幕隐私，可把配置中的 `SHOW_TOAST` 改为 `0`。

个性化 ZIP 含有账号设备凭据，请勿转发。重新下载同一设备的模块并刷入后，新配置会自动替换一次；之后手工修改的配置会保留。

## 运行状态

在 Magisk 中点击模块的“操作”按钮，可以查看是否正在运行、服务器地址和最近日志。日志也位于：

```text
/data/adb/clipboard-sync/clipboard-sync.log
```

通用包需要在以下路径填写配置，然后重启：

```text
/data/adb/clipboard-sync/config.conf
```

## 省电策略

- 手机本机剪贴板：系统回调触发，不定时扫描。
- 电脑发往手机：仅亮屏时保持一个低开销长连接请求；熄屏停止网络检查。
- ROM 不派发剪贴板回调时会在日志中暴露问题，不使用高频轮询掩盖。此类 ROM 后续由单独兼容组件适配。

支持 Android 8.0 及以上。v1.3.0 增加离线最新内容重试、设备状态与同步模式。卸载模块会删除模块保存的配置、令牌、待发送内容和日志。
