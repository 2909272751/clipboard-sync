⚙️ 设备管理

为手机生成一个已经写入服务器地址和专属 Token 的 Magisk 模块。下载后直接在 Magisk 中刷入并重启即可，不需要再编辑配置。

使用方法
1. 下载 ZIP 后不要解压。
2. 打开 Magisk → 模块 → 从本地安装，选择下载的 ZIP。
3. 安装完成后重启手机，模块会自动使用内置配置连接服务器。
下载的 ZIP 包含该设备的专属凭据，请不要转发给其他人。需要给朋友使用时，让朋友登录自己的账号后在这里生成。

💻 Windows 一键客户端

生成免 Python 的 Windows 客户端，服务器地址和专属 Token 已配置好。下载解压后双击“安装并启动.cmd”即可。

使用方法
1. 下载 ZIP 并完整解压到一个文件夹。
2. 双击“安装并启动.cmd”，客户端会立即运行。
3. 程序会自动加入 Windows 开机启动；需要移除时运行“卸载.cmd”。
